# 🌑 Midnight
### Policy Migration Pipeline — Wipro HealthPlan Services

Midnight takes a legacy security policy document and produces a fully formatted, audit-ready `.docx` file matching the official Wipro HPS template — in under 60 seconds.

---

## How It Works

```
Upload legacy policy doc
        ↓
Midnight extracts all content (Groq AI)
        ↓
Python builder formats it to Wipro HPS template
        ↓
Download finished .docx
```

---

## Files

| File | Purpose |
|------|---------|
| `app.py` | Streamlit web interface |
| `hps_policy_migration_builder.py` | Document builder engine |
| `hps_blank_template.py` | Blank template generator |
| `WRITE_IT_OUT_FOR_ME_prompt.py` | Manual migration prompt |
| `requirements.txt` | Dependencies |

---

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## Deploy to Streamlit Cloud

1. Push this repo to GitHub
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Set main file path: `app.py`
5. Deploy

---

## API Key

Midnight uses the Groq API (free tier available at console.groq.com).  
Enter your key in the app — it is never stored.
